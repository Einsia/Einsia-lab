**OpenChronicle** is an open, local-first memory layer for LLM agents. It runs
on your machine, captures structured context from what you're doing, and turns
it into persistent, inspectable Markdown memory — files for projects, tools,
topics, people, organizations, and daily events.

Any tool-calling agent can use it (MCP clients work especially well today). It is
designed as a general memory layer rather than something tied to one model
provider or app: run it fully locally, connect it to any model, and share it
across different agents — private by default.
