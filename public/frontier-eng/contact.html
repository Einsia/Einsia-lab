贡献题目

若您希望为 Frontier-Engineering Bench 贡献新题目，请通过 GitHub 参与协作（仓库 Frontier-Engineering）：

Frontier-Engineering (GitHub)

  • Fork Frontier-Engineering 仓库，在本地克隆您 fork 后的仓库。
  • 按仓库 README 的提交格式添加题目：在对应领域目录下新建任务目录，包含 README.md、Task.md、verification/ 等；分支建议命名为 feat/<领域>/<任务名>。
  • 本地通过 evaluator.py 或 Docker 测试通过后，向上游仓库的 main 分支提交 Pull Request，并在 PR 中说明题目背景、来源及如何运行验证。

完整的贡献指南、提交格式与流程请见 Frontier-Engineering 仓库 README：
